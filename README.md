Muncim is a responsive, interactive web platform that helps job seekers find the best opportunities and assists companies in recruiting efficiently. The project is built with modern frontend technologies and features a smooth user experience.

---

## Features

- **Responsive Design:** Works on desktop, tablet, and mobile devices.
- **Sections:** Home, Who Works, How We Work, Contact, Sign Up.
- **Scroll To Top:** Easy navigation with a fixed scroll-to-top button.
- **Loader Animation:** Beautiful Lottie animation displayed while the site loads.
- **Smooth Scrolling:** Navigation links scroll to the corresponding sections smoothly.
- **Interactive UX:** Designed for both job seekers and companies.

---

##  Tech Stack

- React (Vite)
- CSS Modules
- Lottie (for Loader animation)
- React Icons
- Git & GitHub for version control
